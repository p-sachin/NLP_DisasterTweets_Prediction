####README####

Follow these steps to run the project:
1. Open the project on any IDE with python installed or open the project path in the terminal.
2. select your working environment with python installed.
3. install all the required packages with pip (command: pip install -r requirements.txt)
4. finally run the app.py (python app.py)


Required Packages:
1. Pandas
2. Numpy
3. Sklearn
4. Seaborn
5. Matplotlib
6. Spacy
